Chapter 4.1: Introduction to PCapture-Lib

You now have a solid grasp of vanilla VScript. Now, we introduce a tool that makes all of this faster, easier, and more powerful: PCapture-Lib.

PCapture-Lib is a community-made library of pre-written code that you can include in your projects. It simplifies common tasks and adds brand new functionality that is impossible with standard VScript.

Installation

  1. Download PCapture-Lib.nut from its release page.
  2. Place it in your portal 2/portal2/scripts/vscripts/ folder.
  3. Add this line to the very top of your script file: DoIncludeScript("PCapture-Lib", getroottable()).
[Your Script]
[PCapture-Lib]
(Provides simpler, more powerful functions)
[Vanilla VScript API]
(The basic game functions)
[Portal 2 Engine]
