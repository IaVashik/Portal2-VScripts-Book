Chapter 4.3: Perfect Timing - The ActionScheduler

Standard VScript makes it difficult to schedule complex sequences of events. PCapture-Lib's ActionScheduler solves this with an easy-to-use system that allows for asynchronous actions using the yield keyword. yield tells your function to pause for a set amount of time without freezing the entire game.

New Way (Simple and Readable)

// Turn a light red, then green, then blue, with 1 second between each.
ScheduleEvent.Add("LightSequence", function() {
local light = entLib.FindByName("seq_light")

light.SetColor("255 0 0")
yield 1.0 // PAUSE the function for 1.0 seconds

light.SetColor("0 255 0")
yield 1.0 // PAUSE again

light.SetColor("0 0 255")

}, 0) // Start after 0 seconds delay.

This is a revolutionary feature for creating cinematic sequences, complex puzzle timings, or anything that needs to happen over a period of time.
