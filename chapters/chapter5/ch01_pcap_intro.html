Chapter 5.1: Intro to PCapture-Lib

Congratulations! You have mastered the fundamentals of VScript. You can manipulate entities, manage logic, and even bend the engine to your will with creative hacks. You've reached the limits of what Portal 2's standard scripting tools can offer.

Now, it's time to break those limits.

Why You Need a Library

Think of the standard VScript API as a basic toolbox. It has a hammer, a screwdriver, and a wrench. You can build a lot, but some tasks are incredibly difficult:

  • Your "laser beam" (TraceLine) is blind to players and cubes.
  • Creating timed sequences is a mess of manual delay calculations.
  • Making an object move smoothly from A to B requires complex math in a Think function.
  • Inter-script communication often relies on cluttering the global scope.

PCapture-Lib is a professional-grade power tool kit. It's a massive, community-built library that gives you new, powerful, and intuitive functions to solve these problems and many more. It is the "super-tool" that nearly all modern, complex Portal 2 maps use.

While its size might seem intimidating, PCapture-Lib is designed to be user-friendly, stable, and predictable. Its functions have clear names and behave exactly as you'd expect, making your code cleaner and more powerful.

Installation

Using the library is incredibly simple. It's just another script you include in your project.

  1. Download: Get the latest PCapture-Lib.nut file from the official repository.
  2. Place: Put the file in your portal 2/portal2/scripts/vscripts/ folder alongside your other map scripts.
  3. Include: At the very top of your main logic_script file, add this single line:
// This must be the first line of your script to make all library functions available.
DoIncludeScript("PCapture-Lib.nut", getroottable())

That's it. You are now ready to use the most powerful VScript toolset ever created for Portal 2.
