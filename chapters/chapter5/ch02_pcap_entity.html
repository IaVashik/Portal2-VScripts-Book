Chapter 4.2: Smarter Entities - entLib and pcapEntity

PCapture-Lib replaces the standard Entities object and script handles with its own, enhanced versions: entLib and pcapEntity.

When you find an entity with entLib.FindByName(), you get back a powerful pcapEntity object which allows for much cleaner code through method chaining.

Vanilla VScript vs. PCapture-Lib

Old Way:

local door = Entities.FindByName(null, "mydoor")
if (door)
{
EntFireByHandle(door, "SetColor", "255 0 0", 0, null, null)
EntFireByHandle(door, "Kill", "", 5, null, null)
}

New Way:

local door = entLib.FindByName("mydoor")
if (door)
{
door.SetColor("255 0 0") // Much cleaner!
    .Kill(5)             // Methods can be chained!
}

Some pcapEntity highlights include .SetAlpha(), .SetModelScale(), .Dissolve(), and .EmitSound().
