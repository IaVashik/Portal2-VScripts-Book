Chapter 4.4: The Ultimate Tool - TracePlus

Raycasting (or tracing) is the act of firing an invisible line from one point to another to see what it hits. PCapture-Lib's TracePlus module is a complete replacement for the limited built-in trace functions.

Its crown jewel is TracePlus.PortalBbox(), a trace function that can travel through portals. This is impossible in vanilla VScript and opens up a new dimension of puzzle design.

Example: Portal-Aware Line of Sight

local camera = entLib.FindByName("security_camera_01")
local player = GetPlayerEx() // PCap helper to get the player

if (camera && player)
{
local cameraPos = camera.EyePosition()
local playerPos = player.EyePosition()

// Perform a portal-aware trace from the camera to the player.
local trace = TracePlus.PortalBbox(cameraPos, playerPos, camera)

local hitEntity = trace.GetEntity()

if (hitEntity && hitEntity.isEqually(player))
{
    printl("Camera can see the player!")
}
}
